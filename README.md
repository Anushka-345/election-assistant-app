# 🗳️ Election Process Education Assistant

An AI-powered web application that simplifies election processes using interactive tools like chat assistant, timeline tracker, quiz system, checklist, and calendar reminders. Built to educate and guide users in a simple, engaging way.

---

## ✨ Features

### 🤖 AI Election Buddy
- Chat-based assistant for election queries
- Simple, beginner-friendly explanations
- Neutral and non-partisan responses
- Interactive follow-up questions

---

### 📅 Election Timeline
- Step-by-step process:
  Registration → Campaigning → Voting → Counting → Results
- Visual progress tracking
- Key election date highlights

---

### 🧠 Eligibility Quiz
- Quick 3–5 question quiz
- Checks voter eligibility (age, citizenship, residency)
- Instant result: YES / NO / MAYBE
- Personalized guidance

---

### 🗳️ Voting Checklist
- Registration
- ID & documents
- Polling location
- Voting method
- Election day preparation
- Progress tracking + completion celebration 🎉

---

### ❓ Myth Buster
- Debunks common election myths
- Fact-based explanations
- AI-powered myth checking system

---

### 📆 Calendar Integration
- Add election dates to Google Calendar
- Download iCal (.ics) files
- Key dates:
  - Registration deadline
  - Early voting
  - Absentee ballot deadline
  - Election day

---

### 💬 FAQ Section
- What is EVM?
- What is NOTA?
- How to register?
- What documents are required?
- Where do I vote?

---

## 🛠️ Tech Stack

- React.js (Vite)
- TypeScript
- Tailwind CSS
- Gemini AI (Google Generative AI)
- Framer Motion (animations)
- Calendar APIs (Google Calendar + iCal)
